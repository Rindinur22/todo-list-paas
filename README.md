# To-Do List App — Tugas Cloud Computing

Aplikasi To-Do List sederhana yang dibangun dengan HTML, CSS, dan JavaScript murni.
Dihosting menggunakan Vercel sebagai contoh implementasi **Platform as a Service (PaaS)**.

## Fitur
- ✅ Tambah tugas baru
- ✅ Tandai tugas selesai / belum selesai
- ✅ Hapus tugas
- ✅ Filter: Semua / Selesai / Belum Selesai
- ✅ Statistik tugas
- ✅ Data tersimpan di localStorage (tidak hilang saat refresh)

## Teknologi
- HTML5
- CSS3
- JavaScript (Vanilla)
- Vercel (PaaS Deployment)

## Cara Jalankan Lokal
Cukup buka file `index.html` di browser. Tidak perlu server atau instalasi apapun.

## Deploy ke Vercel
1. Upload folder ini ke GitHub
2. Login ke vercel.com
3. Import repository dari GitHub
4. Klik Deploy
